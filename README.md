# Personal Portfolio — HTML/CSS Final Module Project

A responsive personal portfolio website built with semantic HTML5 and CSS3 as the final project for Coding Temple's HTML/CSS module.

## Project Structure

```
├── index.html          # Main HTML file
├── styles/
│   ├── tokens.css      # CSS custom properties (design tokens)
│   └── main.css        # Component and layout styles
└── images/             # Local image assets
```

## Sections

- **Hero** — Introduction with profile photo and call-to-action
- **Case Studies** — Featured work across Fintech, EdTech, and Pharma
- **Testimonials** — Client testimonials in a responsive grid
- **Recent Work** — Portfolio project previews
- **Contact** — Email, phone, and message form

## Technologies

- HTML5 (semantic elements: `<section>`, `<article>`, `<figure>`, `<blockquote>`, `<nav>`)
- CSS3 with custom properties (design tokens)
- Google Fonts — [Raleway](https://fonts.google.com/specimen/Raleway) (headings) and [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) (body)
- Font Awesome (icons)
- Responsive design with mobile, tablet, and desktop breakpoints

## Running Locally

Open `index.html` directly in a browser — no build step or server required.
